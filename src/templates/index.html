Important Notes

  • To initialize the API

    I used Flask and Flask-PyMongo to create this application in a MacBook.

    -Open and run the 'app.py' file in the 'src' folder

    -Open new terminal and run the following command to start the MongoDB process:

    - If you're in MacOS run: brew services start mongodb-community@6.0

    - If you're in Windows it may be neccessary to install the MongoDB with the following link, I'm not sure. https://www.mongodb.com/try/download/community

    -Once both steps have been done successfully the API should be ready to use as a local host in the port 5000: htttp://localhost:5000/

  • How to send Data

    The API allows anyone to send and receive data in JSON format.

    To send data the user needs to write it in a JSON format like the following:

    {"title":"Game 1", "title 2":"Game 2"}

    you can copy it to an API platform like POSTMAN and send it to the endpoint '/create' with the 'POST' method to create two new projects named 'Game 1' and 'Game 2'.

  • Available Endpoints
    • Create Projects
      • Endpoint: '/create'
      • Method 'POST'
      • Send Format: {"title":"Game 1", "title 2":"Game 2"}
    • See Projects
      • Endpoint: '/projects'
      • Method 'GET'
      • Send Format: -
    • Add Updates to project
      • Endpoint: '/add_update/<project>'
      • Method 'PUT'
      • Send Format: {"update1":"Cards and Ranking", "title 2":"Another Update"} . Type the project's title instead of <project> as it appears in the JSON returned by the '/projects' endpoint
    • Add Relevant Points to Updates
      • Endpoint: '/add_update_description/<project>/<update>'
      • Method 'PUT'
      • Send Format: {"1":"Description1", "2":"Description2"}
    • Search by Project's Date
      • Endpoint: '/search/date/<date>'
      • Method 'GET'
      • Send Format: Type the project's date instead of <date> as it appears in the JSON returned by the '/projects' endpoint
    • Search by Project Title
      • Endpoint: '/search/name/<name>'
      • Method 'GET'
      • Send Format: Type the project's name instead of <name> as it appears in the JSON returned by the '/projects' endpoint
    • Delete SINGLE project
      • Endpoint: '/delete/<project>'
      • Method 'DELETE'
      • Send Format: Type the project's title instead of <project> as it appears in the JSON returned by the '/projects' endpoint
    • Delete ALL project
      • Endpoint: '/delete_all'
      • Method 'DELETE'
      • Send Format: -
